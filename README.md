# Airbnb Market Analysis Dashboard (Tableau)

## Project Overview
This project analyzes Airbnb listing data to evaluate pricing trends 
by bedroom count and ZIP code, along with revenue growth patterns.

## Tools Used
- Tableau
- Data aggregation
- Calculated fields
- Geographic mapping
- Time-series analysis

## Key Insights
- 1-bedroom listings had highest listing volume.
- Significant price variation across ZIP codes.
- Revenue showed steady upward growth over the year.
